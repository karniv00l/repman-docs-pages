# REST API

## Generate a new API Token

In order to start using API you will need to generate `API Token`. In order to do that:

![API Tokens menu](/docs/static/articles/user-profile/menu.png)

1. Click `API Tokens` in user's menu
1. Click `Generate a new API Token`
1. Put token name and click `Generate`
1. You will be redirected back to the list. Now you can copy your newly generated token and start using API.
